Features
Everything watching your inbox

14 real-time checks run on every sender domain the moment an email arrives — before you even open it.

Homograph Attack Detection

Catches invisible character substitutions like paypa1.com vs paypal.com. Unicode confusable analysis runs on every domain automatically.

Unicode · Cyrillic · IDN

Typosquatting Analysis

Levenshtein distance algorithm compares sender domains to legitimate brands. A distance of 1–2 characters triggers instant investigation.

Edit Distance · Fuzzy Match

SPF / DKIM / DMARC Checks

Email authentication checks verified on every message. Failures on SPF, DKIM, or DMARC are weighted heavily in the risk score calculation.

Email Auth · DNS Validation
